# НИССА BAR — Интерактивное меню

## 🚀 Как разместить на GitHub Pages

### Шаг 1: Создание репозитория
1. Зайдите на [github.com](https://github.com)
2. Нажмите **"New repository"**
3. Назовите: `NISSA2026`
4. Выберите **Public**
5. Поставьте галочку **"Add a README file"**
6. Нажмите **"Create repository"**

### Шаг 2: Загрузка файлов
1. В репозитории нажмите **"Add file"** → **"Create new file"**
2. Создайте структуру:
   - `index.html` (в корне)
   - `css/style.css`
   - `js/main.js`
3. Скопируйте содержимое каждого файла
4. Нажмите **"Commit changes"**

### Шаг 3: Включение GitHub Pages
1. Перейдите в **"Settings"** → **"Pages"**
2. В **"Source"** выберите:
   - Branch: **main**
   - Folder: **/ (root)**
3. Нажмите **"Save"**
4. Подождите 2-5 минут

### Шаг 4: Получение ссылки
Ваш сайт: `https://ВАШ_НИК.github.io/NISSA2026/`
